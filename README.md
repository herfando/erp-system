# ERP System

Manufacturing ERP system for managing inventory, production, and sales workflows.

## 🚀 Tech Stack

- Frontend: React (Vite) + TypeScript + Tailwind CSS
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL
- ORM: Prisma
- Infrastructure: Docker (planned)

## 🏭 Core Modules

- Inventory Management (Raw & Finished Goods)
- Production System (BOM, Production Orders)
- Sales Management
- Stock Movement Tracking

## 🎯 Purpose

This project is built to simulate real manufacturing workflows such as those used in factories (plastic, shoes, tires, etc.), inspired by real-world ERP systems like SAP.

## 📌 Status

Early development (core architecture phase)
